# github-claw

这是 **Claw** 的个人 AI 工作空间，基于 GitHub Copilot 构建。

Claw 是长期驻留在这个仓库的 AI 助手，有持续记忆、固定角色和工作方式。

## 快速开始

在新的 Copilot 对话中，用以下提示唤起上下文：

> 请先读取 `AGENTS.md` 和 `MEMORY.md`，继续上次的工作状态。

## 文件说明

| 文件/目录 | 说明 |
|-----------|------|
| `AGENTS.md` | Claw 的身份、工作规则（核心） |
| `MEMORY.md` | 长期持久记忆 |
| `memory/daily/` | 每日工作日志 |
| `memory/tasks/` | 任务状态记录 |
| `memory/context/` | 主题背景知识 |
